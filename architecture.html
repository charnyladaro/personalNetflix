Optimized Video Streaming Architecture (Low Cost)

User
Browser requests video
Cloudflare CDN
Caches static assets (CSS/JS, thumbnails)
Optional caching of videos
Nginx Reverse Proxy
- Handles video streaming (206 Partial Content)
- Gzip/Brotli compression
- Range requests enabled
Flask App (Gunicorn)
- Auth & user logic
- Generates signed URLs
- API for metadata
SQLite (or Postgres)
Stores metadata: users, movies, series
Local Storage
/uploads/videos/
/uploads/thumbnails/
Thumbnail Generator
Creates WebP thumbnails
Done at upload time
Caching Layer
Flask-Caching + Redis (optional)
Speeds up frequent queries

Key Optimizations

  • ✅ Nginx streams videos with range requests (fast seek)
  • ✅ Thumbnails pre-generated as WebP
  • ✅ Cloudflare caches static files globally
  • ✅ Flask only handles auth + API, not heavy video traffic
  • ✅ Use Gunicorn instead of Flask dev server
  • ✅ Optional Redis caching for metadata
