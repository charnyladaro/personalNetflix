üéâ Why Cloudflare Zero Trust is Perfect:

  • No port forwarding needed

üîß Setup Commands:

Cloudflare Tunnel Setup
# Download cloudflared curl -L https://github.com/cloudflare/cloudflared/releases/latest/download/cloudflared-linux-amd64.deb -o cloudflared.deb sudo dpkg -i cloudflared.deb # Login to Cloudflare cloudflared tunnel login # Create tunnel cloudflared tunnel create personal-netflix # Configure tunnel cat > ~/.cloudflared/config.yml << EOF tunnel: personal-netflix credentials-file: ~/.cloudflared/YOUR_TUNNEL_ID.json ingress: - hostname: netflix.yourdomain.com service: http://localhost:5000 - service: http_status:404 EOF # Start tunnel cloudflared tunnel run personal-netflix # Install as service (optional) sudo cloudflared service install
